Loading Spinner
A loading spinner informs the user of an ongoing operation. Only one busy indicator should be shown at a time. The loading input on the component is used to hide and show the element.

Loader element

The loading element is used to display the loading indicator animation. The loading spinner will automatically apply the aria-busy attribute to its parent container when the loading input changes. The loading spinner should always be an immediate child of a container-type element, such as a <div>, <section>, <article>, <h1>, etc.

Usage with other elements

In this example, while loading === true, the parent div will have aria-busy automatically set to true, and the contents of this div will be blurred until loading === false.
